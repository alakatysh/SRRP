# SPPR App

Веб-застосунок Системи підтримки прийняття рішень (СППР) для оцінювання альтернатив за багатьма критеріями за допомогою вагових коефіцієнтів.

Цей репозиторій містить:
- `sppr-backend`: REST API на Express + MySQL
- `sppr-frontend`: Інтерфейс користувача на React + Vite
- `docker-compose.yml` + `init.sql`: локальний запуск MySQL та ініціалізація схеми/даних

## Чому цей проєкт

SPPR App допомагає порівнювати альтернативи (наприклад, варіанти ноутбуків або квартир) з:
- критеріями, що налаштовуються (максимізація або мінімізація)
- вагами для кожного критерію
- матричними значеннями оцінок
- ранжованими аналітичними результатами

## Архітектура

- Фронтенд викликає API за адресою `http://localhost:5000/api`
- Бекенд обслуговує REST-ендпоінти та підключається до MySQL
- MySQL запускається через Docker Compose та ініціалізується з `init.sql`

## Стек технологій

- **Frontend:** React 19, Vite 8, Axios, TailwindCSS 4
- **Backend:** Node.js, Express 5, mysql2, dotenv, helmet, cors
- **Database:** MySQL 8
- **Tooling:** ESLint, Prettier, Nodemon
